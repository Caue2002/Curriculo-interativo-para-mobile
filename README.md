Currículo Interativo 
Visão Geral

Este projeto é um currículo interativo desenvolvido com foco total em dispositivos móveis, utilizando conceitos de jogos 2D para apresentar informações profissionais de forma criativa, dinâmica e intuitiva.

O usuário controla um personagem em um cenário e, ao interagir com diferentes áreas do mapa, visualiza popups com seções do currículo, como dados pessoais, formação acadêmica, experiência profissional, habilidades e redes sociais.

Ao final da navegação por todos os tópicos, é exibido um popup de parabéns, reforçando a conclusão da experiência.

Objetivo do Projeto

Apresentar meu currículo de forma diferenciada e interativa

Demonstrar conhecimentos práticos em HTML, CSS e JavaScript

Aplicar conceitos de responsividade mobile-first

Criar uma experiência fluida, funcional e intuitiva em telas pequenas

Tecnologias Utilizadas

HTML5

CSS3 

JavaScript (Vanilla)

Não utiliza frameworks ou bibliotecas externas, reforçando o domínio da base da web.

Foco em Mobile

Este projeto foi pensado exclusivamente para uso em celulares, com atenção especial aos seguintes pontos:

Layout totalmente responsivo

Controles adaptados para toque

Joystick virtual para movimentação do personagem

Bloqueio de zoom e gestos indesejados

Interface limpa e organizada para telas pequenas

Em desktop, o projeto pode funcionar, mas a melhor experiência é em dispositivos móveis.

Controles

Joystick virtual: controla o movimento do personagem

Ao encostar o personagem em um tópico do mapa:

Um popup é exibido automaticamente

Ao sair da área, o popup é fechado

Não há botões direcionais na tela, garantindo mais espaço visual e melhor usabilidade.

Funcionalidades

Personagem controlável via toque

Popups dinâmicos com conteúdo do currículo

Fechamento automático dos popups ao sair da área

Popup final de parabéns após visualizar todos os tópicos

Layout estável e adaptado para diferentes tamanhos de tela
